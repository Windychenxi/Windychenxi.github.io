Docker 报错 port is already allocated

问题描述

  1. 已经删除了 docker 的进程，使用 docker ps -a 查不到

  2. netstat -anp 可以查到的端口也已经 kill 但还是会报 port is already allocated

解决方法

  1. 查询占用的 docker 映射占用的端口号

  2. kill 掉进程 ID
